Spread Operators

Pass an array to a function: my array = [1,2,3,4,5,6] and myFunction(...myArray){ } allows me to call func with any number of args. Note that myFunction([myArray]) won't work.

Copy an array (but only one level?): colors = ['red', 'blue', 'green']; copy = [...colors] creates a new array, not a reference. Or combine with combinedArr = [...arr1, ...arr2] Equivilent to arr1.concat(arr2)

Objects: const neighbor = {name: 'Bill', age: 45}; const house = {color: white, fence: false}; const combined = {...neighbor, ...house, street: 'Elm'};

Copy an object (but only one level?): const friend = {...neighbor}
